Funções

São ações executadas assim que são chamadas ou em decorrência de algum evento.

Uma função pode receber parâmetros e retornar resultados.

            function ação(parâm) {
                return res
            }

            ação(5)

Exemplo de função para saber se um número é par ou ímpar:

            function parimp (n) {
                if (n%2==0) {
                    return 'Par!'
                } else {
                    return 'Ímpar!'
                }
            }
            
            let res = parimp(9)
            console.log(res)
