Variáveis compostas

São variáveis capazes de armazenar vários valores em uma mesma estrutura. Podem ser chamadas de "vetor" ou "array".

            let num = [8, 3, 7, 6]

O elemento de um vetor é um par que agrupa o espaço da memória, o valor colocado dentro dele e o índice.

O índice ou chave é a identificação da posição do elemento, sempre começando do 0. No exemplo acima, o valor 8 (que é o conteúdo do vetor) está no índice 0, o valor 3 está no índice 1, e assim sucessivamente.

Resumindo: um vetor é uma variável que tem vários elementos, cada elemento é composto por seu valor e por uma chave de identificação.


Acrescentando um elemento

Pode-se acrescentar um elemento em um array indicando o seu índice, por exemplo:

num[4] = 2

Ou usa-se o comando .push:

num.push(2)

num = [8, 3, 7, 6, 2]

Outros comandos:

  • .length = mostra a quantidade de elementos da variável.
  • .sort() = organiza os elementos da variável em ordem crescente.
  • .indexOf(x) = mostra a chave em que se encontra o elemento "x". Caso não exista o elemento retornará o valor -1

Utilizando repetição para exibir os elementos:

  • Ex 01:
    •                 for (let pos=0; pos<num.length; pos++) {
                    console.log(num[pos])
                }

  • Ex 02 (mais simplificado):
    •                 for (let pos in num) {
                    console.log(num[pos])
                }
