Ordem de precedência

  1. ()
  2. **
  3. * / %
  4. + -

Operadores de Atribuição

Atribuição simples

var a = 5 + 3 (a = 8)

var b = a % 5 (b = 3)

var c = 5* b ** 2 (c =45)

var d = 10 - a / 2 (d = 2)

Auto-atribuições

var n = 3

n = n + 4 ou n += 4 (n = 7)

n = n - 5 ou n -= 5 (n = 2)

n = n * 4 ou n *= 4 (n = 8)

n = n ** 2 ou n **= 2 (n = 64)

Incremento e decremento

Pós

var x = 5

x = x + 1 ou x++

x = x - 1 ou x--


Pré

var x = 5

x = x + 1 ou ++x

x = x - 1 ou --x
