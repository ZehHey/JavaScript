Operadores pt 2

Relacionais

  • > - maior que
  • < - menor que
  • >= - maior ou igual
  • <= - menor ou igual
  • == - igual
  • != - diferente

Operadores relacionais terão resultados boleanos (True ou False)

Identidade ou igualdade restrita

5 == 5 = true

5 == '5' = true

5 === 5 = true       5 !== 5 = false

5 === '5' = false     5 !== '5' = true


Lógicos

Ordem de precedência
  • ! - negação
    • ! true = false
    • ! false = true
  • && - conjunção (e)
    • true && true = true
    • true && false = false
    • false && true = false
    • false && false = false
  • || - disjunção (ou)
    • true || true = true
    • true || false = true
    • false || true = true
    • false || false = false

Ternário

"?:"

Ex: 'teste' ? 'true' : 'false'

  • teste = teste lógico a ser realizado
  • true = o que deve acontecer se o resultado do teste for igual a true
  • false = o que deve acontecer se o resultado do teste for igual a false

Ex: média >= 7.0 ? "Aprovado" : "Reprovado"
