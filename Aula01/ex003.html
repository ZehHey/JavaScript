Sinal "+"

Serve como simbolo de adição e também de concatenação.

Pra ser uma adição é necessário ter number + number.

string + string se torna concatenação.

Conversão de variável

O comando windows.prompt recebe valores em string, então é necessário fazer a conversão dessas variáveis.

Comandos: Number.parseInt() ou Number.parseFloat(), mas é possivel apenas usar o comando Number e o próprio script decidirá qual o tipo mais adequado.

Para determinar o número de casas decimais em um float, usa-se o comando n1.toFixed(x), onde x é o número de casas desejado.

Para converter uma variável para string, usa-se os comandos String(n) ou n.toString.

Formatação de strings

Para utilizar o Templates Strings a declaração precisa estar entre crases e a variável dentro do ${}.
