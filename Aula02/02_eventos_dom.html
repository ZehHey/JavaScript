Eventos DOM

É tudo aquilo que possa ocorrer com um elemento, por exemplo: eventos de mouse. Os eventos podem ser configurados na parte html ou diretamente no JavaScript.

Exemplos de eventos de mouse

  • mouseenter - Quando o cursor do mouse fica sobre o elemento;
  • mouseout - Quando o cursor do mouse sai do elemento;
  • mousemove - Quando o cursor do mouse se move sobre o elemento;
  • mousedown - Quando o botão esquerdo do mouse é pressionado;
  • mouseup - Quando o botão esquerdo do mouse é solto;
  • click - Quando o botão esquerdo do mouse é clicado;

Funcão

Esse código é escrito dentro de um bloco, ou seja { }, nomeando esse bloco como function e a ação.
