Iniciando estudos com DOM

O que é DOM?

DOM - Documento Object Model

É o conjunto de objetos dentro do seu navegador, que vai dar acesso aos componentes internos do seu website.


Árvore DOM

Basicamente a árvore DOM começa da raiz, a qual chamamos de window.Então tudo, no código JS, está dentro de um objeto chamado window

Dentro do window teremos outros objetos como: location, document, history, etc.

  • location - diz a localizaçãodo site, a URL, a página atual, qual foi a página anterior;
  • document - o documento atual;
  • history - guarda as informações de navegação;

Dentro do objeto document, existe o objeto html.

window

   |__ location

   |__ document

   |      |__ html

   |            |__ head

   |            |__ body

   |__ history      

Todos esses objetos da árvore DOM são elementos que podem ser selecionados.


Selecionando elementos

São cinco métodos:

  • por Marca: getElementsbyTagName()
    • window.document.getElementsbyTagName('h3')[0]
      • var st = window.document.getElementsByTagName('h3')[0] window.document.write('Subtitulo: ' + st.innerText)
  • por ID: getElementsbyID()
  • por Nome: getElementsbyName()
  • por Classe: getElementsbyClassName()
  • por Seletor:
    • querySelector()
    • querySelectorAll()
      • window.document.querySelector('div#especial').style.color='blue'
